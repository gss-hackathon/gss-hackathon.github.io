Line 明細卡片

設計一個支援 LINE 的詳細資料卡片，常用於顯示某筆資料的詳細內容，內容為欄位與相對應的值。

  • 選擇「文字輸入」或「發送訊息」節點 -> 點擊「訊息」 -> 調整文字訊息為「LINE 明細卡片」節點

1.設定基本內容

製作「LINE 清單卡片」需要先準備一份 JSON 資料

JSON 範例：

{
    "Id": "1001",
    "Name": "王小明",
    "Sex":"男",
    "StartDate": "2003/03/01",
    "Birthday": "1985/09/02",
    "DeptName": "未來平台服務事業處",
    "Tel": "(07)123-4567#11300"
}

2.範例展示

此範例希望展示使用者詳細個人資訊

(1)基本設定

  • 資料需設定標題、內容、資料形態、內容格式
  • 要將資料來源設定到「內容」欄位需要使用 {{$.欄位名稱}}
  • 如使用日期形態則需要額外設定「內容格式」例如 yyyy-MM-dd

(2)調整樣式

根據需求調整樣式，卡片節點雖然可以立即預覽，實際卡片樣式要到 LINE 查看比較準確。
