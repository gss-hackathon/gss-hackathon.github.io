LINE Flex 卡片

Flex Messages是一種可由您自定義卡片版型的訊息格式(JSON格式)，設計時必須符合CSS Flexible Box (CSS Flexbox) 的規範。

相關參考網址:

  • Flex Message elements
  • Flex Message layout
  • Flex Message Simulator：Flex Message線上模擬器。

我們將使用LINE 所提供的Flex Message Simulator線上模擬器快速並有效率進行卡片的設計。

網址：https://developers.line.biz/flex-simulator/

1.Flex Message Simulator線上模擬器介紹

  • 上方功能列：提供NEW、Showcase、Send Message、View as JSON等按鈕功能。

(1)New：可以建立一個新的bubble或是carousel類型的卡片。

(2)Showcase：找不到靈感的話，可以按這個按鈕從中挑選LINE提供的範本，就不用重頭開始設計。

(3)Send Message：將設計好的卡片傳送至您的LINE，可以檢視您所設計的卡片在LINE中實際顯示的效果。

如果您想要接收測試訊息，您必須要在您的LINE加入Flex Message Simulator LINE Official Account

(4)View as JSON：查看卡片的JSON。

  • 左側：卡片預覽。
  • 中間：呈現卡片元件結構，您可以由此點選元件開始操作元件，例：上移、下移、刪除、複製、貼上、重做、復原。
  • 右側：點選中間的元件後，您可以在右側設定元件的屬性。

2.Flex Message elements

Flex Message 是由三種結構所組成的，分別是：container、 blocks (header, hero, body, footer), and components。

(1)Container

Container是Flex Message最上層的結構。包含以下兩種container類型：

(Container的詳細說明，請點選連結參閱LINE官方文件。)

  • Bubble：顯示單張的卡片。
  • Carousel：水平並排顯示多張卡片，使用者可以水平滾動方式瀏覽卡片。

(2)Block

Block是構成Bubble的要素。有以下幾種類型可以使用：

(Block的詳細說明，請點選連結參閱LINE官方文件。)

從上到下顯示的順序是：Header、Hero、Body、Footer。每種類型只能在一個Bubble中使用一次（或也可不使用）。

  • Header：用來顯示卡片的標題或主題。
  • Hero：只能用來顯示圖片。
  • Body：卡片內容區塊。
  • Footer：顯示按鈕或其他補充資訊的區塊。

(3)Component

Component是構成Blocks的元素。以下是可用的Component類型：

(每一種組件的詳細使用方式及限制，請點選連結參閱LINE官方文件。)

  • Box：可以想像是一個容器，Box裡面可以再放入其他的Component。
  • Button：按鈕，當使用者點擊按鈕時，可指定按鈕的執行動作。例如：開啟連結，顯示文字訊息等。
  • Image：圖片組件。
  • Video：影片組件。
  • Icon：圖標組件。（只能使用在baseline box）
  • Text：文字組件。可以指定字體顏色、大小和粗細。
  • Span：可將文字組件細拆成Span，因此可以讓一行文字出現不同的效果。也能分別設定字體顏色、大小、粗細。
  • Separator：分隔線組件。
  • Filler：添加空白空間的組件。

3.如何使用Flex Message

我們使用Flex Message Simulator線上模擬器完成卡片的設計後，只要點選右上的View as JSON按鈕，將整個JSON內容拷貝，回到 C.ai BotBuilder的腳本中，使用「發送訊息」或「文字輸入」節點，文字類型請挑選：LINE Flex卡片，並將拷貝的JSON內容貼進"卡片內容"輸入框中即可。

4.模擬測試

使用模擬測試執行到節點時，可以讓您即時觀看所設計的卡片顯示是否正確。

卡片上的按鈕執行效果，請至LINE實際進行測試。
