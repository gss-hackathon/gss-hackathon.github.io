團隊至少使用一項「叡揚」所提供的數位服務（ iota C.ai 對話服務平台、Vital CRM、Vital BizForm 或 Secure Code Warrior ），
導入應用於校園、企業營運、城市、客服或醫療...等或其他特定場域，開發具有創意及「智慧化」的多元創新應用作品。

若能透過 API 串接，整合多項「叡揚」的數位服務，

依此類推，若同時串接整合其他 3 項服務，至多可加 15%。

    • 一般可用於
      • 組織內部簽核行政使用
        • 例如：請款、請假、出差申請、員工基本資料填寫、報價單、商品資料、利潤分析、設備檢查表等。
      • 亦可作為無帳號填寫或問卷使用
        • 例如：初診基本資料填寫、預約資料、聯繫表單、活動報名表、同意授權書、新人報到資料填寫。
      • 在無帳號填寫上，更可以結合匯入對象資料，達成指定對象問卷填寫
        • 例如：親師聯繫用通知、VIP 問卷、活動滿意度問卷、VIP 資料授權確認書等。
    • 與 Vital CRM 整合為例
      • 在 BizForm 透過引入客戶、公司、商機等欄位建立相關表單。
      • 在 CRM 客戶、公司、商機中，建立具有格式化的表單，並自動帶入相關資料，並於 CRM 中顯示相關表單清單。
      • 將帶有 CRM 客戶資料欄位的 BizForm 表單，填寫後，透過同步按鈕，將客戶資料新增或更新到 Vital CRM 中
      • 會員問卷發送。
      • 出差申請單，將出差記事附上表單中。
    • 與其它系統整合情境
      • 原有系統需要啟動表單簽核，可以將資料帶入 Vital BizForm ，並於簽核通過後更新回系統中。
