LINE清單卡片

設計一個支援 LINE 的列表卡片，常用於需要將資料逐列顯示，清單內容可顯示標題、副標題、描述文字、圖片。

1.設定步驟

(1)加入「LINE 清單卡片」節點

(2)設定標題、文字訊息、資料來源

(3)根據資料內容設定標題、副標題、文字、圖片連結

(4)調整卡片樣式

2.建立 LINE 清單卡片

• 選擇「文字輸入」或「發送訊息」節點 -> 點擊「訊息」 -> 調整文字訊息為「LINE 清單卡片」節點

3.設定基本內容

製作「LINE 清單卡片」需要先準備一份 JSON 資料

JSON 範例：

[
    {
        "Name": "Facebook",
        "Description": "Facebook是源於美國的社群網路服務及社會化媒體網站...",
        "HomeUrl": "https://www.facebook.com/",
        "ImageUrl": "https://imgur.com/qNRwmyV.png"
    },
    {
        "Name": "Google",
        "Description": "Google是源自美國的跨國科技公司...",
        "HomeUrl": "https://www.google.com.tw",
        "ImageUrl": "https://imgur.com/WUySGns.png"
    },
    {
        "Name": "Microsoft",
        "Description": "Microsoft是美國一家跨國電腦科技公司...",
        "HomeUrl": "https://www.microsoft.com/",
        "ImageUrl": "https://imgur.com/hvnJ8To.png"
    }
]

4.範例展示

此範例希望展示 Name、Description、ImageUrl 在每一列，並且使用者點擊資料時連結到　HomeUrl 網址

(1)基本設定

  • 資料需設定標題，副標題、文字、圖片連結為可選
  • 要將資料來源設定到各欄位需要使用 {{$.欄位名稱}}
  • 清單項目點擊動作可以使用 {{$.欄位名稱}} 帶入對應資料

(2)調整樣式

根據需求調整樣式，卡片節點雖然可以立即預覽，實際卡片樣式要到 LINE 查看比較準確。
